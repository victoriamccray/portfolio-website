myCovid Tracker

Experience and Interaction Assignment

Academic assignments often have artifical constraints with real world applicability. The following assignment was given to simulate the iterative design process for a mobile app that would have the potential track covid cases for users. The goals of the assignments were to decrease page content in order for simple and easy extraction of information. This project incorporates rollover and hover interactions.

The prototype was designed using Adobe XD. For this assignment, I was able to leverage user testing in order to integrate consumer usability into my prototype.
